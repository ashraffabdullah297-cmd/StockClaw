## 📌 What is StockClaw?

StockClaw is a straightforward desktop tool made for Windows users who want to study stocks, try trading without real money, test strategies on past data, and get updates through Telegram. It uses smart agents to help with research and analysis but keeps things easy to use.

You do not need programming skills to run or use StockClaw. The app works offline after you download it, except for features that update stock data or send messages on Telegram.

- Research multiple stocks automatically  
- Perform paper trading without risking real money  
- Backtest trading strategies with historical stock data  
- Get alerts and summaries in Telegram  
- Easy setup for Windows users  
- Lightweight and fast  
- No coding required

## 🔧 How to Use StockClaw

1. **Start a new research session**  
   Click *New Research* on the main screen. Enter the stock symbols you want to study (for example, AAPL, MSFT, TSLA).

2. **Run paper trading**  
   Use the *Paper Trade* tab to simulate buying and selling stocks without real money. This lets you test your ideas safely.

3. **Backtest a strategy**  
   Go to the *Backtest* section to apply trading rules on historical data. This helps see if a strategy might work before using it live.

4. **Set up Telegram alerts**  
   Connect your Telegram account by entering your bot token in *Settings > Telegram*. StockClaw will send updates when trading conditions change.

5. **View reports**  
   Use the *Reports* tab to see charts, summaries, and logs of your research results and paper trades.

## 🛠 Troubleshooting

- **App does not start**: Make sure Windows is up to date and you have the .NET Framework installed.  
- **Cannot connect to Telegram**: Verify your bot token and Internet connection.  
- **Download link broken**: Check your internet or visit the GitHub homepage directly and find Releases.  
- **Installation errors**: Run the installer as administrator by right-clicking and choosing “Run as administrator.”
